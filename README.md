# 🐍 Python Mini-Projects

This repository contains a collection of **mini-projects** built in Python. Each project demonstrates practical applications of programming concepts,
ranging from automation and data handling to small interactive applications. The goal is to strengthen Python fundamentals while building recruiter‑friendly artifacts.


# Project Overview
| Project Name                  | Description                                                                 |
|-------------------------------|-----------------------------------------------------------------------------|
| **Audio_book**                | Converts PDF files into audiobooks using text‑to‑speech (pyttsx3).          |
| **Email automation**          | Automates sending internship application emails with attachments via Gmail. |
| **To do list**                | Simple to‑do list app for task management.                                  |
| **Calculator**                | Basic calculator performing arithmetic operations.                          |
| **Fake news detector**        | Detects fake news using machine learning and text classification.           |
| **Guess the number**          | Interactive game where the user guesses a random number.                    |
| **Library management system** | Manages books, members, and transactions in a library.                      |
| **News scraper**              | Scrapes Hacker News top stories and emails them automatically.              |
| **Random (Turtle Graphics)**  | Generates colorful, algorithmic art using Python’s turtle module.           |
| **YouTube video downloader**  | Downloads YouTube videos using Python libraries.                            |


# Skills Demonstrated
- **Automation**: email sending, news scraping, audiobook generation.
- **Data Handling**: dictionaries, sets, file I/O, text processing.
- **Visualization**: turtle graphics for creative coding.
- **Machine Learning**: fake news detection project.
- **Interactive Apps**: games and calculators.
- **Project Documentation**: recruiter‑friendly README files for each project.



# Prerequisites
- **Python 3.7+** installed.
- Libraries used across projects:  
  bash:
  pip install requests beautifulsoup4 smtplib PyPDF2 pyttsx3
- A code editor (e.g., VS Code, PyCharm, or IDLE).


# How to Run
1. Clone the repository:
   bash:
   git clone https://github.com/Paras-Gith/MIni-Project.git
   cd MIni-Project
  
2. Navigate to any project folder:
   bash:
   cd Audio_book
   python audio_book.py
