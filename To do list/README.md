 To-Do List App (Tkinter)
A lightweight desktop To-Do List application built with Python’s Tkinter library.
This app provides a clean interface to manage tasks — add, update, delete, and view them — all in one place.

 Features
- Add Tasks: Quickly add new tasks using the input box.
- Update Tasks: Edit any existing task with a simple dialog.
- Delete Tasks: Remove tasks with confirmation feedback.
- View All Tasks: Display all tasks in a popup window.
- Interactive Listbox: See tasks in real-time as you manage them.
- Exit Button: Close the app gracefully.

 Installation & Setup
- Ensure you have Python 3.x installed.
- Tkinter comes pre-installed with Python, so no extra dependencies are required.
- Clone this repository:
git clone https://github.com/your-username/todo-list-tkinter.git
- Run the app:
python to_do_list_app.py
   Usage
- Type a task in the entry box → click Add Task.
- Select a task in the list → click Update Task or Delete Task.
- Click View Tasks to see all tasks in a popup.
- Click Exit to close the application.
